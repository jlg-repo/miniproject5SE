Displays a set of movies from a local JSON file named "movie.json". Users can filter and sort the list, add movies to a wishlist, mark movies as watched, and download the wishlist as a PDF.

## Features:

- Displays movies in a grid
- Filter by genre, age group, and year
- Sort by release year, IMDb rating, or title
- Wishlist: add/remove movies
- Mark movies as watched
- Download wishlist as a PDF

- React
- Tailwind CSS + DaisyUI (for UI styling)
- jsPDF (for wishlist PDF download option)
